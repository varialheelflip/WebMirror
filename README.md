# WebMirror

WebMirror is a lightweight browser-based camera mirror page.

## Preview

![WebMirror Preview](./.doc/preview.png)

## Features

- Live camera preview in the browser
- Mirror toggle
- Fullscreen mode
- Chinese/English UI switching
- Default language chosen from system/browser language, with manual override remembered locally

## Usage

1. Open [WebMirror.html](./WebMirror.html) in a modern browser such as Edge or Chrome.
2. Allow camera access when prompted.
3. Use the top buttons to toggle mirror mode, enter fullscreen, or switch UI language.
4. The language button shows the current language: `EN` for English, `中文` for Chinese.

## Hotkeys

- `M`: Toggle mirror
- `F`: Toggle fullscreen
- `L`: Toggle language
- `Esc`: Exit fullscreen

## Notes

- The page currently opens the default camera directly.
- Some browsers may restrict access to non-default cameras or device labels based on permission policy.
